This is a lightweight, real-time chat web application based on PIN-based rooms. Users can join or create a chat simply by entering the same PIN—no login, no signup, no session creation, and no database. Messages are temporary and exist only during the active session.

The project focuses on simplicity, privacy, and fast communication.
